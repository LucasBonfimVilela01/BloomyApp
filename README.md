# Este é um projeto feito com React Native e Expo

Link do expo [Expo](https://expo.dev) 

## Get started

1. Instalar dependências

   ```bash
   npm install
   ```

2. Iniciar o app

   ```bash
   npm expo run
   ```

   Ou (instalar o apk do release do app)
